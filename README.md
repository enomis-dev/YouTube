3.  **Activate the Virtual Environment:**

        ```bash
        .\venv\Scripts\activate

        ```

    Navigate to the specific project directory (e.g., `Deep_learning/hand_digits/`) and install the required packages:
    ```bash
    pip install -r requirements.txt
    ```
